babygrad

Build a deep learning library from scratch. Learn how tensors, autograd, and neural networks actually work.

What You'll Build

A PyTorch-like library using only Python and NumPy. By the end, you'll train models on MNIST and understand every line of code that makes it work.

Prerequisites

  • Basic Python (functions, classes, loops)
  • Some familiarity with NumPy is helpful but not required
  • Linear algebra and calculus basics (or willingness to learn)

Setup

# Create project folder
mkdir babygrad-project && cd babygrad-project

# Set up with uv
uv init --name babygrad
uv add numpy
uv add --dev pytest

# Create folder structure
mkdir babygrad examples tests

Your project should look like:

babygrad-project/
├─ .venv/
├─ babygrad/
├─ examples/
└─ tests/
